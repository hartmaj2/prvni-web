Formátování fotografií

  • Za tzv. "character references" je nutne vzdy psat strednik!
  • PNG = Portable Network Graphics a je tzv. "lossless" format
  • PNG je lepsi pouzivat na grafiky, kde neni moc barevnych prechodu
  • JPEG = Joint Photographic Expert Group a je "lossy" format
  • JPEG je nejlepsi pouzivat na fotografie, kde nejsou nutne ostre prechody
  • Do tagu <ul> patri pouze <li> tagy
  • Po dokonceni kazde stranky si kod nechat projit validatorem
  • CSS = Cascading StyleSheets

Classes a idečka

  • Kazdy tag, ktery ma atribut id="attname" je mozne v CSS adresovat pomoci #attname
  • Id ma tu nevyhodu, ze nemuzeme tvorit vice tagu se stejnym id, pokud tyto tagy chceme spolecne nejak formatovat, je dobre pouzit tzv. class, kterou pak v CSS kodu muzeme adresovat jako .classname
  • V CSS mohu jednomu elementu priradit vice classu! Ten, ktery je v CSS definovan nize pak prepise prekryvajici se atributy s tim druhym classem
  • Podtrzitka z linku muzeme odstranit pomoci kodu v CSS napiseme li text-decoration: none;

Speciální formátování

  • Pokud chceme zaoblit okraj ramecku, pouzijeme v CSS prikaz border-radius: 4px;
  • Sirku textu v odstavci muzeme nastavit i relativne vzhledem k sirce stranky napriklad width: 90%;
  • Margin pote muzeme nastavit automaticky aby se dopocital k okrajum pomoci prikazu margin: 20px auto
  • Chceme li, aby prvek zmenil format pouze pokud je nad nim kurzor, pouzijeme v CSS prikaz prvek:hover { ... }
  • V HTML mame dva hlavni typy elementu tzv. block-level elementy a in-line elementy
